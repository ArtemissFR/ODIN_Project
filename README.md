### Comment utiliser ce Projet ?

1. Télécharger le dépots :
   
   ```bash
   git clone https://github.com/ArtemissFR/ODIN_Project
   cd ODIN_Project
   ```

2. Lancer ODIN
   - Lancer ODIN en mode GUI :
   
   ```bash
   .\ODIN_V1.5.ps1
   ```
   
   - Lancer ODIN en mode AUTO (en cours ...):
   ```bash
   .\ODIN_V2.0.ps1
   ```
